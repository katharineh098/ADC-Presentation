Analog-to-Digital Converter (ADC)

Understanding ADC Basics

By Meihui Huang

What is an ADC?

A system that converts an analog signal into a digital signal.

Where are ADCs used?

  1. Microcontrollers
  2. Audio Processing
  3. Sensors
  4. ...

Any application where it is necessary to convert analog signals into digital form.

ADC Process

  1. Sampling
  2. Quantization
  3. Encoding

Sampling

a technique used to capture continuous phenomena, whereby periodic snapshots are taken

Click here for more on the Sampling Theorem.

Sampling Rate

how often an ADC takes samples of the continuous analog

Sampling Theorem

"To reconstruct a signal, the sampling rate must be at least twice the highest frequency component."

Nyquist-Shannon Theorem

\[ f_s \geq 2 f_{max} \]

Quantization

the process of mapping continuous infinite values to a smaller set of decrete finite values

Quantization Error

the difference between an analog signal's actual value and its nearest representable ADC value

Encoding

the process of representing the digital output of the ADC as a binary number

Resolution

the number of distinct digital values that can be used to represent the input signal

A description of how the sequence was sampled and encoded is also needed.
